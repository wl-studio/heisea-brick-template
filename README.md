# brick-template

海西brick组件库，提供快捷的生成snippets扩展。

---
## Features

目前可在vue文件上，对应html/js位置输入快捷代码即可生成列表/表单/比对组件的基础框架。

---
## 快捷代码
#### template: 
    bhlt ---> 列表组件
    bhfm ---> 表单组件
    bhct ---> 比对组件
    bhdl ---> 弹窗组件

#### script:
    bjlt ---> 列表组件
    bjfm ---> 表单组件
    bjct ---> 比对组件
    bjdl ---> 弹窗组件

**Enjoy!**
